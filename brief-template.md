# Creative brief templates (for generating the pre-filled Google Doc)

Generate the brief from one of the layouts below, filling every field known at proposal acceptance
and marking the rest `TBD`. Pass the result as `textContent` to `create_file` with
`contentMimeType: text/plain` — Drive converts it to a Google Doc. Title the doc
`[CODE]-[number] _ Kickoff & Creative Brief`. Use the **General** layout for Social / Campaign /
Website / Branding / Email; use the **Video** layout for Video projects. Delete optional sections
that don't apply.

---

## General — Project Kick-off & Creative Brief

```
DISCOVERY

- Stakeholder contacts: [contacts]
- References: [links]
- Existing assets: [assets]
- Required approvals: [approver(s)]

STRATEGY / BRIEF
- Objective: [objective]
- Target audience: [audience]

- Deliverables list: [list]

OPTIONAL: WEBSITE WORK  (delete if not used)
- Current website / domain · Page(s) needed · Platform / CMS · Hosting / login access ·
  Sitemap / navigation · Content needed · Forms / integrations · SEO · Tracking / analytics ·
  Launch requirements

OPTIONAL: SOCIAL / DIGITAL WORK  (delete if not used)

  Audience / targeting · CTA / destination link · Posting or flight dates · Budget / spend ·
  Reporting metrics

TASK LIST / KICKOFF
- Task template applied: [template name]
- First task: Kickoff meeting — review & finalize creative brief (attendees log time on the subtask).
```
